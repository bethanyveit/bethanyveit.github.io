Shipping Information

On this page you will find information about orders that have yet to be delivered, and a visual map of the whereabouts the orders still need to be delivered to and where orders have already been delivered.

Undelivered Orders

Below you will find a list that shows which orders have still been undelivered. If there is a red box, that means that the order is still undelivered, and if there is a green box, that means that the order has been delivered.

Map of Orders

This is a color coded map of all current orders - delivered and undelivered. You will find a written out key of the colored pins below.

Green = delivered orders

22801 = red (truck one)

22802 = blue (truck two)

22803 = purple (truck three)

22807 = orange (truck four)


Truck Relation

Below you will see what orders will go on what trucks. The boxes in column on the very right hand side correlate with the key of the pin colors on the map. From here, you will be able to match the customers name with which truck their order will be on as well as the zipcode associated with their order.
